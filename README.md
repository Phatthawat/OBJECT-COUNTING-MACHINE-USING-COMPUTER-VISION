# OBJECT-COUNTING-MACHINE-USING-COMPUTER-VISION
โปรเจ็กนี้เป็นโปรเจ็กจบภาคการศึกษาระดับปริญญาตรี คณะวิศวกรรมศาสตร์ สาขาอิเล็กทรอนิกส์อัจฉริยะ #01
